Our Pitch​

Inherent safety-guarantees with the Rust programming language.

Ability to use all the normal Rust tooling ‒ clippy, crates.io, IDE’s, etc.

Interoperability with Solidity contracts.


 


Clear migration path for graduating to a parachain.

Established industry standard for the compiler target.

Native to Polkadot. Complements your Substrate parachain perfectly.

Where can I deploy ink! v6 contracts?​

ink! v6 is still in development, it can't be deployed to production yet.

We plan to release ink! 6.0.0-alpha in February and will update this section with testnet information then.

You can follow Parity's plans for deploying the ink! execution engine (pallet-revive) to Kusama and Polkadot on the Polkadot Forum.
